# Web App with Firebase Anonymous Authentication

This project is a simple web app that lets users sign in anonymously using Firebase and then interact with Firestore. It’s useful for prototypes, quick demos, or apps where you don’t need full user accounts but still want per-user data.

# Features

- Anonymous sign-in using Firebase Authentication
- Firestore read and write support
- Automatic session persistence
- Clean and minimal frontend
- Easy to extend for full authentication later
